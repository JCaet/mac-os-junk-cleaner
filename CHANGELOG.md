# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-01-01

### Added
- Initial stable release of `macos-junk-cleaner`.
- CLI commands: `scan` and `clean`.
- Support for common macOS junk files: `.DS_Store`, `._*`, `.Spotlight-V100`, etc.
- Dry-run mode by default for `clean` command.
- Rich-formatted output for better readability.
- Basic test suite.
- Development setup for both `pip` and `uv`.
- Configured automated versioning with `python-semantic-release`.

### Changed
- Minimum Python version increased to 3.9 to ensure build compatibility.
- Version bumped to 1.0.0 to match project state and git tags.
